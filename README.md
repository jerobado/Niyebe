Niyebe
---

_Niyebe_, simple password generator for the forgetful

### Usage

```
$ niyebe -l 16      # generate random string with 16 characters
TJ1q!5@!VTo40eUe
```

### Features & Benefits
- generate random custom password

### Build Instruction
```
$ cmake -H. -Bbuild
$ make -C build
$ ./build/bin/niyebe
```
